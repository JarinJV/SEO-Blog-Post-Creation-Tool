# SEO Blog Post Creation Tool

## Overview
This project is an AI-based tool that automates the creation of SEO-optimized blog posts using trending or best-selling products from e-commerce platforms.

## What This Project Does
- Scrapes trending or best-selling products from e-commerce websites
- Identifies relevant SEO keywords using keyword research tools
- Generates 150–200 word blog posts with keyword integration
- Publishes content to platforms like WordPress or Medium

## Purpose
To streamline SEO content creation by automating product research, keyword optimization, and blog writing.
